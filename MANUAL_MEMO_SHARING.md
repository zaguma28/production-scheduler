# メモ・図形の共有とkintone設定について

本アプリケーションでは、作業者PCと管理者PCの間でメモや図形（スタンプ）を共有するために **kintoneのデータベース** を使用します。

## 概要
- 作成したメモや図形は、自動的にkintoneアプリにレコードとして登録されます。
- これにより、離れた場所にある作業者のPCでも同じ図形が表示されます。

## kintone側の設定（推奨）

メモや図形データは、kintone上では以下のようなレコードとして保存されます。
- **品名**: `MMO` または `SHAP`
- **備考**: `{"x":100, "y":200...}` のようなデータ文字列

これらが通常のスケジュールと混ざって表示されるのを防ぐため、kintone側で**一覧の表示設定**を変更することを強く推奨します。

### 設定手順：メモ・図形を隠すビューの作成
1. kintoneアプリの **設定（歯車アイコン）** を開きます。
2. **「一覧」** タブを選択し、新しい一覧を作成（またはデフォルト一覧を編集）します。
3. **絞り込み条件** に以下を設定します：
   - `品名` `≠（等しくない）` `MMO`
   - **かつ**
   - `品名` `≠（等しくない）` `SHAP`
4. **保存** して **「アプリを更新」** します。

これにより、kintoneの画面上ではスケジュールのみがきれいに表示され、裏側では図形データもしっかり共有されます。

## トラブルシューティング

### 「kintoneへの追加に失敗しました」エラーが出る場合
図形を追加・コピーする際にエラーが出る場合は、**APIトークンの権限** を確認してください。

1. kintoneアプリ設定 > **APIトークン** を開く。
2. 使用しているトークンに **「レコード閲覧」「レコード追加」「レコード編集」「レコード削除」** のすべてのチェックが入っているか確認する。
   - ※特に「レコード追加」がないとコピー機能でエラーになります。
3. 設定を変更したら必ず **「アプリを更新」** してください。
